Spiral Matrix II

LeetCode #59


Description

Given an integer n, generate a square matrix filled with elements from 1 to n^2 in spiral order.

For example,

Given n = 3,

You should return the following matrix:

[
 [ 1, 2, 3 ],
 [ 8, 9, 4 ],
 [ 7, 6, 5 ]
]


My Thoughts

This is similar to Spiral Matrix. So I would like to solve this problem recursively too. The only different is that I’m not going to push value in matrix into answer list but to make a matrix.


Solution

class Solution(object):
    def generateMatrix(self, n):
        """
        :type n: int
        :rtype: List[List[int]]
        """
        if n==0: return []
        self.matrix=[]
        self.count = 1
        
        for i in range(n):
            self.matrix.append([None]*n)
        
        def recSpiralOrder(row, col, level):
            if row <= 0: return
            if col <= 0: return
            
            if col > 1 and row > 1:
                for i in range(level, col -1 +level):
                    self.matrix[level][i] = self.count
                    self.count += 1

                for i in range(level, row -1 +level):
                    self.matrix[i][col -1 +level] = self.count
                    self.count += 1

                for i in range(col -1 +level, level, -1):
                    self.matrix[row -1 +level][i] = self.count
                    self.count += 1

                for i in range(row -1 +level, level, -1):
                    self.matrix[i][level] = self.count
                    self.count += 1
                
            elif col == 1 and row == 1:
                self.matrix[level][level] = self.count
                self.count += 1
                
            elif col == 1:
                for i in range(level, row -1 +level):
                    self.matrix[i][col -1 +level] = self.count
                    self.count += 1
                self.matrix[row-1+level][level] = self.count
                self.count += 1
                    
            elif row == 1:
                for i in range(level, col -1 +level):
                    self.matrix[level][i] = self.count
                    self.count += 1
                self.matrix[level][col -1 +level] = self.count
                self.count += 1
                
            return recSpiralOrder(row-2, col-2, level+1)
        
        recSpiralOrder(n, n, 0)
        return self.matrix
