Container With Most Water

LeetCode #11


Description

Given n non-negative integers a1, a2, …, an, where each represents a point at coordinate (i, ai). n vertical lines are drawn such that the two endpoints of line i is at (i, ai) and (i, 0). Find two lines, which together with x-axis forms a container, such that the container contains the most water.

Note: You may not slant the container and n is at least 2.


My Thoughts

It is easy to do with two pointers, left and right. Calculate the rectangle between left and right first and update max area, and then try to move left or right pointer:

* If left height is less than right height, move left pointer until left height >= right height.
* Vice versa


Solution

class Solution(object):
    def maxArea(self, height):
        """
        :type height: List[int]
        :rtype: int
        """
        max_area = 0
        lh = height[0]
        rh = height[len(height)-1]
        i=0
        j=len(height)-1

        while i<j:
            tmp = min(lh,rh)*(j-i)
            max_area = max(tmp,max_area)
            if lh<rh:
                while i<j and height[i]<=lh:
                    i=i+1
                if i<j:
                    lh=height[i]
            else:
                while i<j and height[j]<=rh:
                    #print("in")
                    j=j-1
                if i<j:
                    rh=height[j]

        return max_area
