Spiral Matrix

LeetCode #54


Description

Given a matrix of m x n elements (m rows, n columns), return all elements of the matrix in spiral order.

For example,

Given the following matrix:

[
 [ 1, 2, 3 ],
 [ 4, 5, 6 ],
 [ 7, 8, 9 ]
]

You should return [1,2,3,6,9,8,7,4,5].


My Thoughts

At first, I think about using a pointer to do that. So at first it goes right and until the next point does not exist, it changes its direction. And when the pointer goes through a point, I’ll change the value to None. So I’ll not go to that point again. However, I have to change the input in this case.

Therefore, I think I have to do it in a different way. I would like to try to solve this problem recursively by breaking this problem into subproblems. For every subproblems, I just have to go through all the points that are at the edge of the matrix. And after I go through all the points, just call the recursive function again with smaller matrix.


Solution

class Solution(object):
    def spiralOrder(self, matrix):
        """
        :type matrix: List[List[int]]
        :rtype: List[int]
        """
        if len(matrix)==0: return []
        if len(matrix[0])==0: return []
        
        self.ans = []
        
        def recSpiralOrder(matrix, row, col, level):
            if row <= 0: return
            if col <= 0: return
            
            if col > 1 and row > 1:
                for i in range(level, col -1 +level):
                    self.ans.append(matrix[level][i])

                for i in range(level, row -1 +level):
                    self.ans.append(matrix[i][col -1 +level])

                for i in range(col -1 +level, level, -1):
                    self.ans.append(matrix[row -1 +level][i])

                for i in range(row -1 +level, level, -1):
                    self.ans.append(matrix[i][level])
                
            elif col == 1 and row == 1:
                self.ans.append(matrix[level][level])
                
            elif col == 1:
                for i in range(level, row -1 +level):
                    self.ans.append(matrix[i][col -1 +level])
                self.ans.append(matrix[row-1+level][level])
                    
            elif row == 1:
                for i in range(level, col -1 +level):
                    self.ans.append(matrix[level][i])
                self.ans.append(matrix[level][col -1 +level])
                
            return recSpiralOrder(matrix, row-2, col-2, level+1)
        
        recSpiralOrder(matrix, len(matrix), len(matrix[0]), 0)
        return self.ans
