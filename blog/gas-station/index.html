Gas Station

LeetCode #134


Description

There are N gas stations along a circular route, where the amount of gas at station i is gas[i].

You have a car with an unlimited gas tank and it costs cost[i] of gas to travel from station i to its next station (i+1). You begin the journey with an empty tank at one of the gas stations.

Return the starting gas station’s index if you can travel around the circuit once, otherwise return -1.

Note: The solution is guaranteed to be unique.


My Thoughts

If I don’t care about the time complexity, then I can just try every index as the start point and see if I can complete the circuit. If yes, then return this index, otherwise, return -1. It will take O(n^2) to find the answer.

To speed up, I would like to think it as a greedy problem. Then I have to start at index 0 and never come back again. It also means that every index i would only pass through one time.

So start from index 0, there are two conditions:

  • If gas[0] is larger than or equal to cost[0], it means I have enough gas to move to next station, so index 0 may still be start point of the circle.
  • Otherwise, it means there are no enough gas to move to the next station if I start from index 0, so index 0 is not a correct start point. I can try to use index 1 as a new start point.

However, the above can only work at index 0, so if I want to do index i, I need three variables:

  • “sum” as a sum of the gas from the start point.
  • “answer” which is the temporary start point.
  • “total” which is total sum of the gas[i] - cost[i].

So when it comes to index i, there are two conditions:

  • If “sum” += gas[i]-cost[i] >= 0, it means there are enough gas to move to next station from start point “ans”.
  • Else, it means there are not enough gas to move, so I have add current “sum” to “total” and reset “sum”. And update the start point “ans” to i+1.

If all the gas stations is run through, check if “total” is larger than 0, if not, it means that it is impossible to find a start point which can complete circuit. Or just return “ans”.


Solution

class Solution(object):
    def canCompleteCircuit(self, gas, cost):
        """
        :type gas: List[int]
        :type cost: List[int]
        :rtype: int
        """
        sum = 0
        ans = 0
        total = 0
        for i in range(len(gas)):
            sum += gas[i]-cost[i]
            if sum<0:
                total += sum
                sum = 0
                ans = i+1
        total+=sum
        if total<0: return -1
        return ans
