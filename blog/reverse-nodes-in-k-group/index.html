Reverse Nodes in k-Group

LeetCode #25


Description

Given a linked list, reverse the nodes of a linked list k at a time and return its modified list.

k is a positive integer and is less than or equal to the length of the linked list. If the number of nodes is not a multiple of k then left-out nodes in the end should remain as it is.

You may not alter the values in the nodes, only nodes itself may be changed.

Only constant memory is allowed.

For example,

Given this linked list: 1->2->3->4->5

For k = 2, you should return: 2->1->4->3->5

For k = 3, you should return: 3->2->1->4->5

My Thoughts

Because I have to reverse part of the linked list from ith to (i+k)th, I think I need a helper function reverse(preTail, nextHead). PreTail is the tail of previous part of the linked list and nextHead means the beginning of the next part of linked list. And this helper function will reverse this part of linked list (from preTail.next to nextHead), and return the tail of reversed linked list.

The reason why I need previous tail and next head is that I have to linked all reversed linked list together.

For example, if I get a linked list a->b->c->d->e and k = 2:

Take second run for example, then the current linked list would look like b->a->c->d->e. So preTail = a, nextHead = e, and preTail.next = c. After I reversed c->d to d->c, I have to link this part to the whole linked list. So preTail.next should be d and c.next should be nextHead. Then return c as next preTail.

Solution

# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution(object):
    def reverseKGroup(self, head, k):
        """
        :type head: ListNode
        :type k: int
        :rtype: ListNode
        """
        if((head == None) or (k <= 1)):
            return head
        dummy = ListNode(0)
        dummy.next = head
        preTail = dummy
        
        count = 0
        while(head != None):
            count += 1
            if(count == k):
                preTail = self.reverse(preTail, head.next)
                head = preTail.next
                count = 0
            else:
                head = head.next
        return dummy.next
        
        
        return dummy.next
    def reverse(self, preTail, nextHead):
        head = preTail.next
        tail = preTail.next
        previous = None
        current = head
        while(current != nextHead):
            nextNode = current.next
            current.next = previous
            previous = current
            current = nextNode
        head = previous
        preTail.next = head
        tail.next = nextHead
        return tail
