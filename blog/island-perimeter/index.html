Island Perimeter

LeetCode #463


Description

You are given a map in form of a two-dimensional integer grid where 1 represents land and 0 represents water. Grid cells are connected horizontally/vertically (not diagonally). The grid is completely surrounded by water, and there is exactly one island (i.e., one or more connected land cells). The island doesn’t have “lakes” (water inside that isn’t connected to the water around the island). One cell is a square with side length 1. The grid is rectangular, width and height don’t exceed 100. Determine the perimeter of the island.

Example:

[[0,1,0,0],
 [1,1,1,0],
 [0,1,0,0],
 [1,1,0,0]]

Answer: 16


My Thoughts

It is easy to do it directly with two nested for loop. Just like solution 1.

However, it will go through some unnecessary cells. So I want to solve this problem with stack and only go to the necessary parts. See solution 2.

So I will use a nested for loop to find the first land first, and push this position into stack.

Then while the stack is not empty, do the following steps:

  1. Pop the first element from stack.
  2. Change that value into 2.
  3. Check it’s top, left, right and down and add the perimeter if needed.
  4. Push the position into stack if that position is also a land.


Solution

Solution 1
class Solution(object):
    def islandPerimeter(self, grid):
        """
        :type grid: List[List[int]]
        :rtype: int
        """
        perimeter=0
        for i in range(len(grid)):
            for j in range(len(grid[i])):
                if grid[i][j]==1:
                    #up
                    if i==0:
                        perimeter = perimeter+1
                    else:
                      if grid[i-1][j]==0:
                          perimeter = perimeter+1
                    #left
                    if j==0:
                        perimeter = perimeter+1
                    else:
                      if grid[i][j-1]==0:
                          perimeter = perimeter+1
                    #right
                    if j==(len(grid[i])-1):
                        perimeter = perimeter+1
                    else:
                      if grid[i][j+1]==0:
                          perimeter = perimeter+1
                    #down
                    if i==(len(grid)-1):
                        perimeter = perimeter+1
                    else:
                      if grid[i+1][j]==0:
                          perimeter = perimeter+1
        return perimeter
Solution 2
class Solution(object):
    def islandPerimeter(self, grid):
        """
        :type grid: List[List[int]]
        :rtype: int
        """
        perimeter=0
        stack = []
        
        def findFirstLand():
            for i in range(0, len(grid)):
                for j in range(0, len(grid[i])):
                    if grid[i][j]==1:
                        stack.append((i,j))
                        grid[i][j] = 2
                        return
        findFirstLand()
        
        while len(stack) > 0:
            x, y = stack.pop()
            # top
            if x == 0 or grid[x-1][y] == 0:
                perimeter += 1
            elif grid[x-1][y] == 1:
                stack.append((x-1,y))
                grid[x-1][y] = 2
            # left
            if y == 0 or grid[x][y-1] == 0:
                perimeter += 1
            elif grid[x][y-1] == 1:
                stack.append((x,y-1))
                grid[x][y-1] = 2
            # right
            if x == (len(grid)-1) or grid[x+1][y] == 0:
                perimeter += 1
            elif grid[x+1][y] == 1:
                stack.append((x+1,y))
                grid[x+1][y] = 2
            # down
            if y == (len(grid[x])-1) or grid[x][y+1] == 0:
                perimeter += 1
            elif grid[x][y+1] == 1:
                stack.append((x,y+1))
                grid[x][y+1] = 2
            
            
        return perimeter
