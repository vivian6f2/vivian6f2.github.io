Path Sum IV

LeetCode #666


Description

If the depth of a tree is smaller than 5, then this tree can be represented by a list of three-digits integers.

For each integer in this list:

1 The hundreds digit represents the depth D of this node, 1 <= D <= 4. 2 The tens digit represents the position P of this node in the level it belongs to, 1 <= P <= 8. The position is the same as that in a full binary tree. 3 The units digit represents the value V of this node, 0 <= V <= 9.

Given a list of ascending three-digits integers representing a binary with the depth smaller than 5. You need to return the sum of all paths from the root towards the leaves.

Example 1:

Input: [113, 215, 221]
Output: 12
Explanation: 
The tree that the list represents is:
    3
   / \
  5   1

The path sum is (3 + 5) + (3 + 1) = 12.

Example 2:

Input: [113, 221]
Output: 4
Explanation: 
The tree that the list represents is: 
    3
     \
      1

The path sum is (3 + 1) = 4.


My Thoughts

This is related to Path Sum, Path Sum II and Path Sum III. I will store tree into dictionary with key = (level, position). For every node (i,j), it’s parent should be (i-1, j/2), and it’s child should be (i+1, j*2) and (i+1, j*2+1). So I can move and add the path value into dictionary first. And then check into dictionary again, to find the leaf node which doesn’t have left and right child. And add those value.

Solution

class Solution(object):
    def pathSum(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        if len(nums) == 0:
            return 0
        if len(nums) == 1:
            return nums[0]%10
        tree = {}
        tree[(0,0)] = nums[0]%10
        
        for i in range(1, len(nums)):
            level = nums[i]/100 - 1
            position = (nums[i]%100)/10 - 1
            value = nums[i]%10
            if (level-1, position/2) in tree:
                tree[(level, position)] = tree[(level-1, position/2)] + value
        
        pathSum = 0
        
        for i in range(1, len(nums)):
            level = nums[i]/100 - 1
            position = (nums[i]%100)/10 - 1
            if (level+1, position*2) not in tree and (level+1, position*2+1) not in tree:
                pathSum += tree[(level, position)]
        return pathSum
