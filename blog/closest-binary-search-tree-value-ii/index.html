Closest Binary Search Tree Value II

LeetCode #272


Description

Given a non-empty binary search tree and a target value, find k values in the BST that are closest to the target.

Note:

  • Given target value is a floating point.
  • You may assume k is always valid, that is: k â‰¤ total nodes.
  • You are guaranteed to have only one unique set of k values in the BST that are closest to the target.

Follow up:

Assume that the BST is balanced, could you solve it in less than O(n) runtime (where n = total nodes)?


My Thoughts

It takes too much time to do it brute force. That is pushing all elements into array and find the k nearest values.

I would try to do it with two stacks, one for larger numbers and another for smaller numbers. And run through the tree to push numbers into them.

  • Smaller stack: Pass the tree inorder (left child, current node, right child) and push the value that is smaller than target.
  • Larger stack: Pass the tree with the order of right child, current node, and left child. And also push the value that is larger than target.

After that, just run a while loop with k > 0. And pop the value in smaller or larger stack where the value is closer to target, and k–.


Solution

# Definition for a binary tree node.
# class TreeNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution(object):
    def closestKValues(self, root, target, k):
        """
        :type root: TreeNode
        :type target: float
        :type k: int
        :rtype: List[int]
        """
        self.small = []
        self.large = []
        
        def fillSmallStack(root):
            if root == None: 
                return
            fillSmallStack(root.left)
            if root.val > target: 
            	# all the value after root.right is larger than target
                return
            self.small.append(root.val)
            fillSmallStack(root.right)
        def fillLargeStack(root):
            if root == None: 
                return
            fillLargeStack(root.right)
            if root.val <= target: 
            	# all the value after root.left is smaller than target
                return
            self.large.append(root.val)
            fillLargeStack(root.left)
            
        fillSmallStack(root)
        fillLargeStack(root)
        ans = []
        while k > 0:
            k -= 1
            if len(self.large) == 0:
                ans.append(self.small.pop())
            elif len(self.small) == 0:
                ans.append(self.large.pop())
            elif (target - self.small[-1]) < (self.large[-1] - target):
                ans.append(self.small.pop())
            else:
                ans.append(self.large.pop())
        return ans
