Sort Transformed Array

LeetCode #360


Description

Given a sorted array of integers nums and integer values a, b and c. Apply a function of the form f(x) = ax^2 + bx + c to each element x in the array.

The returned array must be in sorted order.

Expected time complexity: O(n)

Example:

nums = [-4, -2, 2, 4], a = 1, b = 3, c = 5,

Result: [3, 9, 15, 33]

nums = [-4, -2, 2, 4], a = -1, b = 3, c = 5

Result: [-23, -5, 1, 7]


My Thoughts

It is easy to do it brute force, compute the answer for all numbers and sort the answer array. But it takes O(n log n) time, I need to do it faster.

So then take a look at the function, it is ax^2 + bx + c. The graph of this function may look like:

  • If a > 0, a curve with minimum point at x = -b/2a.
  • If a < 0, a curve with maximum point at x = -b/2a.
  • If a = 0, a line and I need to consider if b is larger than 0 or not.

So I think I can solve this problem with two pointers.

  • If a = 0:
    • If b >= 0, calculate the answer and return the array directly.
    • Else, calculate the answer, and return the reverse array.
  • If a > 0, use two pointers, left and right:
    1. Compare the answer at left and right, append larger one into array first and move left or right.
    2. Keep doing it until right is less than left.
    3. Return reversed array.
  • If a < 0, also use two pointers, left and right:
    1. Compare the answer at left and right, append smaller one into array and move left or right.
    2. Keep doing it until right is less than left.
    3. Return array.


Solution

class Solution(object):
    def sortTransformedArray(self, nums, a, b, c):
        """
        :type nums: List[int]
        :type a: int
        :type b: int
        :type c: int
        :rtype: List[int]
        """
        def func(x):
            return a*x*x + b*x + c
        answer = []
        
        if a == 0:
            if b >= 0:
                for i in range(0, len(nums)):
                    answer.append(func(nums[i]))
                return answer
            else:
                for i in range(0, len(nums)):
                    answer.append(func(nums[i]))
                return answer[::-1]
        elif a > 0:
            left = 0
            right = len(nums)-1
            while (left <= right):
                if func(nums[left]) >= func(nums[right]):
                    answer.append(func(nums[left]))
                    left += 1
                else:
                    answer.append(func(nums[right]))
                    right -= 1
            return answer[::-1]
        else:
            left = 0
            right = len(nums)-1
            while (left <= right):
                if func(nums[left]) <= func(nums[right]):
                    answer.append(func(nums[left]))
                    left += 1
                else:
                    answer.append(func(nums[right]))
                    right -= 1
            return answer
