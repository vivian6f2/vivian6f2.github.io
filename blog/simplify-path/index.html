Simplify Path

LeetCode #71


Description

Given an absolute path for a file (Unix-style), simplify it.

For example,

path = “/home/”, => “/home”

path = “/a/./b/../../c/”, => “/c”


My Thoughts

I will use stack to solve this problem. And there are some conditions that I will push or pop the stack:

  • If I get nothing after “/”, do nothing.
  • If I get “pathname/”, push “pathname/” into stack.
  • If I get “./”, do nothing.
  • If I get “../”, pop stack one time.


Solution

class Solution(object):
    def simplifyPath(self, path):
        """
        :type path: str
        :rtype: str
        """
        path_list = path.split('/')

        stack = []
        for value in path_list:
            if value == '' or value=='.':
                continue
            elif value== '..':
                if len(stack)>0:
                    stack.pop()
            else:
                stack.append(value)
                
        if len(stack)==0: return "/"
        else:
            str = ''
            for value in stack:
                str = str + "/" + value
            return str
