Binary Tree Zigzag Level Order Traversal

LeetCode #103


Description

Given a binary tree, return the zigzag level order traversal of its nodes’ values. (ie, from left to right, then right to left for the next level and alternate between).

For example:

Given binary tree [3,9,20,null,null,15,7],

    3
   / \
  9  20
    /  \
   15   7

return its zigzag level order traversal as:

[
  [3],
  [20,9],
  [15,7]
]


My Thoughts

It is similar to Binary Tree Level Order Traversal. However, I have to do it with zigzag order, so it’s impossible to do it with queue anymore. And I also need an extra flag to know which way I’m going to do now.

So take the following tree as an example:

    3
   / \
  1   2
 / \ / \
4  56   7

The correct order is:

[
  [3],
  [2, 1],
  [4,5,6,7]
]

So I would need a deque and a flag.

Let me define the flag first:

  • If flag is <-, it means my deque should be inserted from left and removed from right. And handle the right child first.
  • If flag is ->, it means my deque should be inserted from right and removed from left. And handle the left child first.

So let me do this problem with the tree above:

  1. For level 0, the flag is <-, insert 3 into deque from left. Deque = [3]
  2. Then for level 1, the flag is ->, pop 3 from left and insert 1, 2 from right. Deque = [1, 2]
  3. Then for level 2, the flag is <-, pop 2 from right and insert 7, 6 from left. Deque = [6, 7, | 1]. And then pop 1 from right and insert 4, 5 from left. Deque = [4, 5, 6, 7].
  4. Then for level 3, the flag is ->, pop 4, 5, 6, 7 from left.


Solution

# Definition for a binary tree node.
# class TreeNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution(object):
    def zigzagLevelOrder(self, root):
        """
        :type root: TreeNode
        :rtype: List[List[int]]
        """
        if root == None: return []
        
        Ans = []
        Deque = [root]
        leftToRight = True
        
        while(len(Deque) > 0):
            size = len(Deque)
            smallList = []
            for i in range(0, size):
                if leftToRight:
                    node = Deque.pop(0)
                    smallList.append(node.val)
                    if node.left:
                        Deque.append(node.left)
                    if node.right:
                        Deque.append(node.right)
                else:
                    node = Deque.pop()
                    smallList.append(node.val)
                    if node.right:
                        Deque.insert(0, node.right)
                    if node.left:
                        Deque.insert(0, node.left)
            leftToRight = not leftToRight
            Ans.append(smallList)
        return Ans
