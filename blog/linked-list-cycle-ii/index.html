Linked List Cycle II

LeetCode #142


Description

Given a linked list, return the node where the cycle begins. If there is no cycle, return null.

Note: Do not modify the linked list.

Follow up: Can you solve it without using extra space?


My Thoughts

To check if I have already been to a node, I can add a hash map to store every node I gone through. However, if I use a hash map, I need O(n) extra space. So I want to change the way I think about it.

Then two pointers may be a possible way to solve this problem. Just make one pointer move slow (one step everytime) and another pointer move fast (two step everytime). And keep running to see if following two conditions happen:

  • If fast == None, there is no cycle in this linked list.
  • If fast == slow, there is a cycle in this linked list.

If there is not a cycle, just return None. While there is a cycle, I have to return the node where the cycle begins. So I can make slow go back to head again, and move slow one step and fast one step everytime until slow meets fast. And that node is the beginning of the cycle.

Following is the explanation of finding beginning of the cycle:

Assume:

  • slow and fast pointer meet at step K.
  • the length of cycle is R.
  • So, 2K (fast) - K (slow) = n * R, K = n * R.
  • the distance between start of the list and start of the cycle is S.
  • the distance between start of the cycle and first meet node is M.
  • So, S = K - M.
  • S = K - M = n * R - M = (n-1) * R + (R-M), and take n = 1 (first meet).
  • So, S = R - M. It means that the distance between start of the list and start of the cycle is equal to the length of cycle minus M.

Therefore, if I start a new pointer from head to the start of cycle = S steps. Which is equal to another pointer start from first meet node and go to the start of cycle = R - M.


Solution

# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution(object):
    def detectCycle(self, head):
        """
        :type head: ListNode
        :rtype: ListNode
        """
        if head==None or head.next==None: return None
        
        
        slow = head
        fast = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow==fast:
                break
        if slow==fast:
            slow = head
            while slow!=fast:
                slow =slow.next
                fast = fast.next
            return slow
        
        return None
