Search in Rotated Sorted Array

LeetCode #33


Description

Suppose an array sorted in ascending order is rotated at some pivot unknown to you beforehand.

(i.e., 0 1 2 4 5 6 7 might become 4 5 6 7 0 1 2).

You are given a target value to search. If found in the array return its index, otherwise return -1.

You may assume no duplicate exists in the array.


My Thoughts

I think this is a binary search problem. So everytime I can seperate it into two parts with middle index. The following are the conditions I have thought about (mid = (low + high)/2):

  • If nums[mid] == target, return mid
  • Else if nums[mid] < target, I cannot make sure which part I have to search now, so there are more conditions:
    • If nums[mid] > nums[high], it means the rotation happens after mid. So before mid, all the numbers are smaller than target. Go search the array after mid => low = mid + 1.
    • Else if nums[mid] < nums[high], it means that the rotation happens before mid. So I need to check the value of nums[high] to see where target should be.
      • If nums[high] < target, it means target should appear before mid.
      • If nums[high] > target, it means target can only appear after mid.
  • Else if nums[mid] > target, I also cannot make sure which part to search now:
    • If nums[mid] > nums[high], it means the rotation is after mid. So there may be some numbers smaller than nums[mid] == target before and after mid.
      • If nums[high] < target, it means target should appear before mid.
      • If nums[high] > target, it means target should appear after mid.
    • If nums[mid] < nums[high], it means the rotation happens before mid. So all the numbers smaller than nums[mid] is before mid => high = mid.


Solution

class Solution(object):
    def search(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: int
        """
        if len(nums)==0: return -1
        if nums[0]==target: return 0
        
        low = 0
        high = len(nums)-1
        
        while (low<high):
            mid = low + (high-low)/2
            if(nums[mid]<target):
                if nums[mid]<nums[high]:
                    if nums[high]<target:
                        high = mid
                    elif nums[high]>target:
                        low = mid+1
                    else: return high
                elif nums[mid]>nums[high]:
                    if nums[high]==target: return high
                    else: low = mid+1
            elif(nums[mid]>target):
                if nums[mid]<nums[high]:
                    if nums[high]==target: return high
                    else: high = mid
                elif nums[mid]>nums[high]:
                    if nums[high]<target:
                        high = mid
                    elif nums[high]>target:
                        low = mid+1
                    else: return high
            else:
                return mid
                
        return -1
