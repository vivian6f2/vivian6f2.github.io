Decode Ways

LeetCode #91


Description

A message containing letters from A-Z is being encoded to numbers using the following mapping:

'A' -> 1
'B' -> 2
...
'Z' -> 26

Given an encoded message containing digits, determine the total number of ways to decode it.

For example,

Given encoded message “12”, it could be decoded as “AB” (1 2) or “L” (12).

The number of ways decoding “12” is 2.


My Thoughts

It is a dynamic programming problem. So just build the table correctly and I can get the answer from that table.

I will need a function isLetter(s) which will return if s is a letter or not. And there are only two conditions:

  1. If the first character in s is ‘0’, then it is not a letter.
  2. Transform s into integer, if the number is from 1~26, then it is a letter.

Then I can build my table with isLetter function.

  1. My table, count[n] = how many different ways I can decode from s[0] to s[n-1].
  2. count[n] =
    • If s[n-1] is letter and s[n-2]s[n-1] is letter -> count[n-1] + count[n-2]
    • If only s[n-1] is letter -> count[n-1]
    • If only s[n-2] is letter -> count[n-2]
    • Else -> 0
  3. Boundary condition:
    • count[1] =
      • If s[0] is letter -> 1
      • Else -> 0
    • count[0] = 1
  4. Return count[len(s)-1]


Solution

class Solution(object):
    def numDecodings(self, s):
        """
        :type s: str
        :rtype: int
        """
        if len(s) == 0: return 0
        
        def isLetter(s):
            if s[0]=='0': return False
            return int(s) > 0 and int(s) <= 26
        
        if len(s) == 1 and isLetter(s[0]):
            return 1
        elif len(s) == 1:
            return 0
        
        count = []
        if isLetter(s[0]):
            count.append(1)
        else:
            count.append(0)
            
        if isLetter(s[1]) and isLetter(s[0]+s[1]):
            count.append(count[0] + 1)
        elif isLetter(s[0]+s[1]):
            count.append(1)
        elif isLetter(s[1]):
            count.append(count[0])
        else:
            count.append(0)
            
        for i in range(2, len(s)):
            if isLetter(s[i]) and isLetter(s[i-1]+s[i]):
                count.append(count[i-1]+count[i-2])
            elif isLetter(s[i-1]+s[i]):
                count.append(count[i-2])
            elif isLetter(s[i]):
                count.append(count[i-1])
            else:
                count.append(0)
        
        return count[len(s)-1]
