3 Sum Smaller

LeetCode #259


Description

Given an array of n integers nums and a target, find the number of index triplets i, j, k with 0 <= i < j < k < n that satisfy the condition nums[i] + nums[j] + nums[k] < target.

For example, given nums = [-2, 0, 1, 3], and target = 2.

Return 2. Because there are two triplets which sums are less than 2:

[-2, 0, 1]
[-2, 0, 3]


My Thoughts

It is similar to 3 Sum problem, with a little change:

  • Sort array as usual.
  • Run for loop from index = 0 to last element in array.
  • Create a two sum function which works with two pointers, left and right:
if nums[index] + nums[left] + nums[right] < target:
	totalCount += right - left
	left ++
if nums[index] + nums[left] + nums[right] >= target:
	right --


Solution

class Solution(object):
    def threeSumSmaller(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: int
        """
        self.count = 0
        def twoSumSmaller(index, target):
            left = index + 1
            right = len(nums) - 1
            while left < right:
                if (nums[index] + nums[left] + nums[right]) < target:
                    self.count += right - left
                    left += 1
                else:
                    right -= 1
        
        nums.sort()
        for i in range(0, len(nums)-2):
            twoSumSmaller(i, target)

        return self.count
