Binary Tree Postorder Traversal

LeetCode #145


Description

Given a binary tree, return the postorder traversal of its nodes’ values.

For example:

Given binary tree {1,#,2,3},

return [3,2,1].

Note: Recursive solution is trivial, could you do it iteratively?


My Thoughts

This is a similar question to Binary Tree Inorder Traversal and Binary Tree Preorder Traversal.

Postorder = left child, right child, current node.

At first it is hard to figure out how to do it. But actually, it is similar to preorder traversal.

Preorder = current node, left child, right child.

Postorder = left child, right child, current node.

If I change the order of left and right child of preorder, it will look like: current node, right child, left child. And it is only the reverse of postorder.

So if I can get two conclusions:

  1. If I only need right child, left child, current node, it means that I can just push number into answer list from the beginning instead of from the back.
  2. Then I need to change the order of right and left child to make the order lookes like left child, right child, current node which is exactly postorder. Just go down to the right most child first and insert value from the beginning of the answer list. And start popping out the stack and move to left child one time and keep moving to the right most child again. And don’t forget to insert value to the answer list.


Solution

# Definition for a binary tree node.
# class TreeNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution(object):
    def postorderTraversal(self, root):
        """
        :type root: TreeNode
        :rtype: List[int]
        """
        ans = []
        stack = []
        node = root
        while node:
            ans.insert(0, node.val)
            stack.append(node)
            node = node.right
        
        while stack:
            node = stack.pop()
            left = node.left
            while left:
                ans.insert(0, left.val)
                stack.append(left)
                left = left.right
            
        return ans
